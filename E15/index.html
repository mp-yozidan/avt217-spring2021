Instructions

  1. Open Adobe XD.
    • If you do not have Adobe XD, open a digital art or design software available to you.
    • The software should have the ability to draw basic shapes.
  2. Create a new file with the dimensions of 1000 by 1000 pixels.
  3. Make the artboard fill color pure white (hex code #FFFFFF).
    • If using an illustration software, use the bucket fill tool to make your lowermost layer #FFFFFF.
    • All further instructions will be given for Adobe XD and similar programs.
  4. Select the rectangle tool.
  5. Draw a square 500 by 500 pixels.
  6. Switch back to the arrow tool.
  7. Drag the square to the exact center of the artboard.
  8. In the sidebar, change the square's fill color to pure magenta (#FF00FF).
  9. In XD, there is a default border option. Make sure it's activated for the square. The border should be 1 pixel in width with a color of #707070.
  10. Select the ellipse tool.
  11. Draw a circle with the dimensions of 500 by 500 pixels.
  12. Switch back to the arrow tool.
  13. Place this circle in the top-right corner so that its edges align with the edges of the artboard.
  14. In the sidebar, change the circle's fill color to pure cyan (#00FFFF).
  15. Make sure the circle is in front of the magenta square.
  16. Make sure the circle has no border.
  17. Select the ellipse tool again.
  18. Draw a circle with the dimensions of 200 by 200 pixels.
  19. Switch back to the arrow tool.
  20. Drag this smaller circle on top of the bigger one and align it in the center of the larger circle.
    • If you're having trouble, the coordinates for this smaller circle in XD should be X=650 and Y=150.
  21. Make sure the smaller circle is in front of the large one.
  22. In the sidebar, change the small circle's color to pure yellow (#FFFF00).
  23. Make sure the small circle has no border.
  24. Go to File > Export > All Artboards.
  25. Save the drawing as a PNG file to a folder of your choice.
  26. Open your school email and create a new message addressed to mpippert@masonlive.gmu.edu
  27. Attach the PNG to the email and send. You're done!
